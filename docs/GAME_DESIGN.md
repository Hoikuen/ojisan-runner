# おじさんランナー — GAME DESIGN

## コンセプト
健康診断／メタボから必死に逃げるおじさんのエンドレスランナー。コミカル・可愛い・必死。
「ただ走る」に"逃げる理由"を足して共感の笑いを作る。

## 最初のゴール（vertical slice・図形プレースホルダーで）
絵はまだ描かず、次が成立する最小ループを完成させる：
- おじさん（四角でよい）が自動で右に走る
- タップ／クリックでジャンプ（できればスライドも）
- 障害物が流れてくる → 当たるとゲームオーバー
- 距離（スコア）が増える・ベスト距離を保存
- リトライできる
これが「面白い」と確認できてから、絵を作り込んで差し替える。

## 操作
- タップ／クリック：ジャンプ
- （任意）下フリック／長押し：スライド

## 要素（絵は後で差し替え）
- 障害物＝誘惑（高カロリー）：ラーメン・ビール・唐揚げ・ソファ・自販機
- アイテム＝ヘルシー：野菜・水・ダンベル・青汁（加速・無敵・スコア）
- 追手：体重計・白衣の医者・メタボ判定（後ろから迫る／捕まると終了）

## アート方向性（差し替え時）
2〜3頭身デフォルメ。腕ブン回し＆足バタバタの誇張した走り（4〜6コマ）。汗・効果線でコミカル。必死顔が可愛い。

## 技術メモ
1レーン＋ジャンプ／スライドのみ＝物理は軽い（重力・床めり込み等の危険地帯を避ける）。
Phaser 3 + Vite を想定。詳細は ~/Developer/games/PHASER3_CONSTRAINTS.md。
